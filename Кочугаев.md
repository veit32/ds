# 1. Правило суммы и произведения. Принцип Дирихле
## Принцип Дирихле

> Если число объектов больше, чем количество мест, то существует клетка, где расположено более одного объекта

## Доказательство

Пусть имеется $n$ мест и $m$ объектов, причем 
$$m>n$$ 
$k_i$ - количество объектов на итом месте. Пусть на каждом месте не более одного объекта:
* $k_1 \leq 1$ - число объектов на 1-ом месте 
* $k_2 \leq 1$ - число объектов на 2-ом месте
* $\dots$
* $k_n \leq 1$ - число объектов на последнем месте
Найдем количество всех объектов. 
$$m = k_1 + k_2 + \dots + d_n \leq 1 + 1 + \dots + 1 = n$$
или 
$$m \leq n$$
что противоречит условию $m > n$, следовательно принцип Дирихле верен

## Правило суммы и произведения

Пусть первый объект можно выбрать $m$ способами, второй объект - $n$ способами

> **Правило суммы**: Число способов выбрать первый ЛИБО второй предмет равно $(n + m)$ 

**Пример:** На столе лежат 3 яблока и 5 груш. Сколькими способами можно взять фрукт со стола?
**Решение:** $(3 + 5)= 8$

---

> **Правило произведения:** Число способов выбрать первый, ЗАТЕМ второй предмет равно $(m \cdot n)$

**Пример:** Из города A в город B можно добраться тремя способами, их города B в город C - четыремя способами. Сколькими способами нам можно добраться из города A в город C? (Ходить одними и теми же дорогами несколько раз и возвращаться  - нельзя)
**Решение:** Вначале нам нужно добраться из города A в город B, щатем из города B в город C: $(3\cdot4) = 12$

Данные правила можно применять и в случаях, когда объектов больше, чем два.

# 2. Основные комбинаторные объекты: перестановки, размещения, сочетания и их подсчет

Итак,
> **Выборка** - это процесс выбора элемента из множества по определенным правилам

Существуют следующие типы выборок:
- **Без повторений**
	- **Сочетания (Неупорядоченная выборка)**: Выбор $k$ элементов из $n$ элементного множества без учета порядка: 
	  $$C_n^k=\frac{n!}{k!(n-k)!}$$
	  Например: Пусть дано множество $A = \{1, 2, 3\}$. Все сочетаний данного множества по $2$: $<1, 2>,<1, 3>,<2, 3>$. Сочетания $<1, 2>$ ; $<2, 1>$ - для нас одинаковы
	- **Размещения (Упорядоченная выборка):** Выбор и упорядочивание $k$ элементов из $n$ элементного множества: 
	  $$A_n^k = \frac{n!}{(n-k)!}$$
	  Например: Пусть дано множество $A = \{1, 2, 3\}$. Все размещения данного множества по $2$: $<1, 2>,<1, 3>,<2, 3>,<2, 1>,<3, 1>,<3, 2>$. размещения $<1, 2>$; $<2, 1>$ - для нас различны
- **С повторениями**
	- **Сочетания (Неупорядоченная выборка)**: Выбор $k$ элементов из $n$ элементного множества без учета порядка: 
	  $$\overline{C_n^k}=С^k_{n+k-1}=\frac{(n+k-1)!}{k!(n-1)!}$$
	- **Размещения (Упорядоченная выборка):** Выбор и упорядочивание $k$ элементов из $n$ элементного множества: 
	  $$\overline{A}_n^k = n^k$$

Отдельно выделю **Перестановку**: $$P_n=n!$$
По факту это частный случай **Размещения без повторений** 
$$P_n=A_n^n=n!$$

# 5. Принцип включения - исключения

Пусть имеется множество $U$, причем $|U|=N$. Пусть имеются свойства $a_1 \dots a_n$, и каждый элемент множества $U$ имеет или не имеет свойство $a_i$

> Имеет место следующее соотношение:
> $$N(\overline{a} _ 1 \dots \overline{a} _ n ) = N - \sum_i N(a_i) + \sum_{i < j}N(a_i a_j) - \sum_{i < j < k} N(a_i a_j a_k) + \dots + (-1)^n N(a_1 \dots a_n)$$
> Где $N(a_1, \dots a_n)$ - кол-во элементов с заданными свойствами, $N(\overline{a}_1, \dots, \overline{a_n})$ - кол-во элементов, НЕ имеющее свойства $a_1, \dots, a_n$
